# Welcome to the Workshop

Building a Coding Agent with Semantic Kernel

---

## What We'll Build

A fully functional coding agent that can:

- Execute shell commands
- Read and write files
- Follow custom instructions
- Track tasks and maintain focus

---

## Workshop Structure

1. **Foundation** - Understanding agents
2. **Agent Loop** - Building the core
3. **Tools** - Adding capabilities
4. **Context Engineering** - Improving performance
5. **Conclusion** - Next steps

- Basic C# knowledge
- .NET 8 SDK installed
- An IDE (VS Code or Visual Studio)
- Azure OpenAI or OpenAI API access

- Context engineering techniques
